# Meine Visitenkarte

## Beschreibung  
Eine einfache, interaktive Visitenkarten-Webseite, erstellt mit HTML, CSS und JavaScript.
Das Projekt ist Teil meines Lernprozesses im Bereich Frontend-Entwicklung.

## Features
Saubere HTML-Struktur
Modernes Card-Layout mit CSS
Responsive Zentrierung
Interaktive "Mehr über mich" Funktion (JavaScript Toggle)

## Technologien
HTML5
CSS3
JavaScript (Vanilla)
